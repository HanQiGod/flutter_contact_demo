# flutter_contact_demo

基于 Flutter 的通讯录功能示例，覆盖两类典型场景：

- 原生联系人选择器：使用 `flutter_native_contact_picker_plus`
- 完整通讯录读取与搜索：使用 `contactos`

## 已实现内容

- Android / iOS 通讯录权限配置
- 系统原生联系人选择
- 选择联系人中的特定手机号
- iOS 多联系人选择入口
- 读取完整通讯录列表
- 本地姓名 / 手机号 / 邮箱搜索
- 按需加载联系人头像
- 写入一个示例联系人到系统通讯录
- 权限拒绝与永久拒绝提示
